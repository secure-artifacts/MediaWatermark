# 视频批量水印工具（PyQt6 + FFmpeg）

一个本地桌面工具，用于给视频批量添加 **图片水印** 或 **文字水印**。底层使用 FFmpeg，速度和兼容性都比较好。

## 功能
- 批量添加水印：支持拖拽添加文件/文件夹
- 图片水印：透明度、大小（按视频宽度百分比）、位置（含自定义坐标）
- 文字水印：透明度、字号（按视频高度百分比）、位置（含自定义坐标）
- 输出：同目录或指定目录；格式 mp4/mkv/mov
- 进度显示：单文件进度 + 总体进度
- 可取消：终止当前 ffmpeg 进程后停止
- 智能调色： 开启后，程序将通过 AI 采样视频背景色，自动为您选择对比度最高的文字颜色及描边，确保在任何背景下水印都清晰可见。 

## 依赖
- Python 3.9+
- `pip install -r requirements.txt`
- 系统需安装 `ffmpeg` + `ffprobe`，并加入 PATH（命令行可直接运行 `ffmpeg -version`）或者自行把`ffmpeg` + `ffprobe`放入Tools下

## 运行
```bash
python main.py
```



> 说明：因为会调用外部 ffmpeg，可选择在目标机器上安装 FFmpeg，或把 ffmpeg/ffprobe 放到同目录并修改代码使用相对路径。
